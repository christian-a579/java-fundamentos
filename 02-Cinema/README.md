# 🎬 Sistema de Cinema

Projeto desenvolvido em **Java** para praticar os fundamentos da linguagem através de um sistema simples de gerenciamento de filmes em um cinema.

---

## 📖 Sobre o projeto

A aplicação é executada via console e permite ao usuário realizar operações básicas de cadastro e gerenciamento de filmes utilizando uma lista em memória (`ArrayList`).

Este projeto foi desenvolvido durante meus estudos para a certificação **Oracle Certified Associate (OCA Java)**.

---

## 🚀 Funcionalidades

- 📋 Listar filmes cadastrados
- 🔍 Buscar filme pelo título
- ➕ Cadastrar novo filme
- ❌ Remover filme
- 🚪 Encerrar aplicação

---

## 🛠️ Conceitos praticados

Durante o desenvolvimento deste projeto foram utilizados os seguintes conceitos de Java:

- Classes e Objetos
- Métodos
- Métodos estáticos
- ArrayList
- Scanner
- Estruturas condicionais (`if` / `else`)
- Laços de repetição (`for`)
- Encapsulamento básico de dados
- Manipulação de listas


