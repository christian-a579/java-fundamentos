# 📚 Sistema de Biblioteca

Projeto desenvolvido em **Java** para praticar os fundamentos da linguagem através de um sistema simples de gerenciamento de livros em uma biblioteca.

---

## 📖 Sobre o projeto

A aplicação é executada via console e permite ao usuário realizar operações básicas de cadastro e gerenciamento de livros utilizando uma lista em memória (`ArrayList`).

Este projeto foi desenvolvido durante meus estudos para a certificação **Oracle Certified Associate (OCA Java)**.

---

## 🚀 Funcionalidades

- 📋 Listar livros cadastrados
- 🔍 Buscar livro pelo título
- ➕ Cadastrar novo livro
- ❌ Remover livro
- 🚪 Encerrar aplicação
